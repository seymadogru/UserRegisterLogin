Projenin github linki:https://github.com/seymadogru/UrlShortener.git

Lütfen Postman'de şu şekilde requestler göndererek çıktıları alınız.

POST: http://localhost:8083/generate
Body: {"url": "https://github.com/seymadogru"} (JSON tipinde)

Ardından tarayıcınızda localhost:8083/'shortlink' yaparak orijinal linke gidebilirsiniz.
